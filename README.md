# Sztuczna Inteligencja

Repozytorium zawiera materialy do laboratoriow z przedmiotu **Sztuczna Inteligencja**: notatniki Jupyter z kolejnymi zadaniami oraz plik danych wykorzystywany w cwiczeniach.

## Zawartosc

- `lab_1.ipynb`
- `lab_2.ipynb`
- `lab_3.ipynb`
- `lab_4.ipynb`
- `lab_5.ipynb`
- `Audiobooks_data.csv`
- `lab_6` pod adresem: https://github.com/szymon-tulodziecki/news_ai

## Wymagania

- Python 3.10+
- Jupyter Notebook lub VS Code z rozszerzeniem Jupyter
- Biblioteki: `numpy`, `pandas`, `matplotlib`, `scikit-learn`

## Uruchomienie

1. Utworz srodowisko wirtualne i zainstaluj zaleznosci:

	```bash
	python -m venv .venv
	./.venv/Scripts/activate
	pip install numpy pandas matplotlib scikit-learn jupyter
	```

2. Otworz wybrany notatnik i uruchamiaj komorki po kolei.

## Dane

Plik `Audiobooks_data.csv` zawiera przykladowy zestaw danych. Notatniki zakladaja, ze plik znajduje sie w katalogu glownym repozytorium.

## Struktura pracy

- Kazdy notatnik jest samowystarczalny i zawiera opis krokow.
- Wyniki i wykresy sa zapisywane w notatnikach, bez osobnych plikow wyjsciowych.
